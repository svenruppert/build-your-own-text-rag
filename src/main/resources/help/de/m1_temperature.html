Zweck. Steuert, wie deterministisch das Sampling des Modells ist. Niedrige Werte legen das Modell auf den wahrscheinlichsten Token fest; hohe Werte erlauben mehr Variation.

Bereich. 0,0 bis 1,0 typisch (Ollama akzeptiert bis 2,0). Standard 0,7 für Chat.

Hinweis. Für sachliche Fragen oder Code unter 0,3 bleiben, damit das Modell auf eine Antwort festgelegt wird. Für kreatives Schreiben gibt 0,7–0,9 mehr Abwechslung. Über 1,0 wird die Ausgabe oft inkohärent.
