Zweck. Die eindeutige Bezeichnung, die diesen Eintrag im Speicher identifiziert. Wird im Treffer-Grid neben dem Score angezeigt und als Schlüssel beim Aktualisieren oder Löschen verwendet. Ein zweites Hinzufügen mit gleicher Id wird abgelehnt.

Werte. Beliebiger nicht-leerer String. Leer lassen für Auto-Generierung: auto-<uuid8>.

Hinweis. Eine aussagekräftige Id (doc-42, faq-ollama) verwenden, wenn ein Eintrag später gezielt gefunden oder ersetzt werden soll; leer lassen bei der Erkundung – die Auto-Id ist ausreichend.
