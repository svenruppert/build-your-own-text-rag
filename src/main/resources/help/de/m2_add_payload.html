Zweck. Der im Treffer-Grid angezeigte lesbare Text für diesen Eintrag. Er wird neben dem Vektor gespeichert, durchläuft aber nicht das Einbettungsmodell – der eingebettete Text ist das separate Feld "Text to embed".

Werte. Beliebiger Text. Standardmäßig wird der eingebettete Text verwendet, wenn leer gelassen.

Hinweis. Für schnelle Tests leer lassen – der gleiche Text wird sowohl eingebettet als auch angezeigt. Überschreiben, wenn die zu suchende Repräsentation sich von der anzuzeigenden unterscheidet: z.B. einen kurzen Anfrage-Satz einbetten, aber den vollständigen Abschnitt, eine URL oder einen Titel anzeigen.
