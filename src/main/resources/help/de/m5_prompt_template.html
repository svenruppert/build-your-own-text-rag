Zweck. Welche Prompt-Vorlage die abgerufenen Chunks umhüllt. Die strikte Vorlage weist das Modell an, zu verweigern, wenn der Kontext eine Antwort nicht unterstützt.

Optionen.

  • aus -- SimpleGroundedPromptTemplate. Das Modell kann auf parametrisches Wissen zurückgreifen.
  • ein -- StrictRefusalPromptTemplate. Das Modell wird angewiesen, "Ich weiß es nicht" zu antworten, wenn die Chunks die Antwort nicht enthalten.

Hinweis. Für compliance-sensible Demos einschalten; für explorative Gespräche ausschalten. Eine strikte Vorlage erhöht die Ablehnungsrate, reduziert aber Halluzinationen.
