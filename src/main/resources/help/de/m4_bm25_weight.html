Zweck. Das Gewicht, das auf den BM25-Score in der gewichteten-Score-Fusion angewendet wird. Zusammen mit dem Vektorgewicht – die beiden summierten sich per Konvention üblicherweise auf 1,0, obwohl der Code das nicht erzwingt.

Bereich. 0,0 bis 1,0. Standard 0,4.

Hinweis. Für schlüsselwortlastige Korpora (Code, Logs, technische Referenzen) erhöhen; für Fließtext, bei dem Synonyme und Paraphrasen dominieren, senken.
