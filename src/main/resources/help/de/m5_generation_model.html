Zweck. Welches Ollama-Modell die fundierte Antwort aus den abgerufenen Chunks schreibt.

Werte. Jedes chat-fähige Modell in Ollama; die Liste wird von /api/tags abgerufen. Standard bevorzugt gemma4:e4b, dann andere gemma-, llama- oder qwen-Familien-Modelle.

Hinweis. Kleine Instruct-Modelle antworten schnell und bleiben im Kontext, wenn der Prompt straff ist. Reasoning-Modelle geben Denk-Tokens in das Seitenpanel aus und liefern bessere Antworten bei Multi-Chunk-Fragen, zum Preis von Sekunden bis Minuten Latenz.
