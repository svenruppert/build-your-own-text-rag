Zweck. Welcher Index die Abfrage beantwortet. Vektor ist semantisch; BM25 ist lexikalisch; Hybrid verbindet beide.

Optionen.

  • Vektor -- Nächste-Nachbarn über Einbettungen. Stark bei Paraphrasen, schwach bei seltenen Eigennamen.
  • BM25 -- Term-Frequenz-Bewertung. Stark bei exakten Begriffen und Abkürzungen, blind für Synonyme.
  • Hybrid (RRF) -- verbindet beide Ranglisten per reziproker Rangfusion.
  • Hybrid (gewichtet) -- lineares Mischen normierter Scores.

Hinweis. Hybrid ist der sichere Standard für gemischte Abfragen; reiner Vektor gewinnt bei gut strukturiertem Fließtext; reines BM25 gewinnt bei Code oder Logs.
